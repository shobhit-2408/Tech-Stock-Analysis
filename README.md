# 📊 Stock Market Analysis Dashboard

An end-to-end **Stock Market Analysis Project** combining **Data Preprocessing, Exploratory Data Analysis (EDA), and an Interactive Dashboard** built using **Python, Streamlit, Plotly, and Seaborn**.

---

## 🚀 Live Demo

👉 **[View Interactive Dashboard](https://tech-stock-analysis-db.streamlit.app/)**

---

## 📌 Project Overview

This project analyzes historical stock data of major tech companies (**Amazon, Apple, Google, Microsoft**) to uncover trends, patterns, and relationships.

It consists of two major parts:

### 🔹 1. Data Preprocessing & EDA (Jupyter Notebook)

* Data cleaning and formatting
* Handling missing values
* Converting date columns to datetime
* Feature engineering (daily returns, moving averages)
* Exploratory Data Analysis using visualizations

### 🔹 2. Interactive Dashboard (Streamlit App)

* Real-time data visualization
* User-controlled company selection
* Dynamic charts and KPIs

---

## 📌 Features

* 📈 **Closing Price Trend Visualization**
* 📊 **Moving Averages (10, 20, 50 Days)**
* 📉 **Daily Returns Analysis**
* 📅 **Resampled Data (Monthly, Quarterly, Yearly)**
* 🔥 **Correlation Heatmap between Stocks**
* ⚡ **Interactive Dashboard with Sidebar Controls**
* 📊 **KPI Metrics (Latest, Average, Max Price)**
